# ABCU-Advising-Assistance-Software
A C++ course planner that loads course data, stores it using data structures, sorts courses alphanumerically, and allows users to search and display prerequisites. This project demonstrates my understanding of algorithms, runtime analysis, and clean program design.

What was the problem you were solving in the projects for this course?

The main problem I was solving in this course was how to properly store, organize, and display course information using data structures. I needed to build a program that could read course data from a file, sort the courses in the correct order, and allow the user to search for a specific course and see its prerequisites. The bigger goal was learning how to choose the right data structure and understand how it affects performance and memory usage.

How did you approach the problem? Consider why data structures are important to understand.

I approached the problem by focusing on how the course data should be stored so it could be sorted and searched easily. I chose a data structure that made organizing and accessing the information efficient. This project showed me that understanding data structures is important because they impact performance, memory use, and how cleanly a program works.

How did you overcome any roadblocks you encountered while going through the activities or project?

When I ran into roadblocks, I broke the problem into smaller parts and tested each section of the program separately. I reviewed my logic, checked for errors step by step, and adjusted my approach when something was not working. Taking it one piece at a time helped me fix issues with file reading, sorting, and searching without getting overwhelmed.

How has your work on this project expanded your approach to designing software and developing programs?

This project expanded my approach by making me think more about planning before coding. Instead of just writing code and hoping it works, I focused more on how the data should be structured and how each part of the program connects. It helped me see that good design decisions at the beginning make the program easier to build and improve later.

How has your work on this project evolved the way you write programs that are maintainable, readable, and adaptable?

This project helped me focus more on writing clean and organized code instead of just making it work. I paid more attention to using clear structure, meaningful naming, and separating tasks into functions. It showed me that writing readable code makes it easier to fix bugs, make changes, and adapt the program in the future.
